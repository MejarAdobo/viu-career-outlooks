# About - VIU Career Tool

This tool is designed to help future students connect VIU programs and credentials to prospective career paths, as well as offer a deeper insight into the career outlook for BC and Canada.

Project designed and developed by student web developer Ryan Paranich, VIU summer co-op 2022.

# Developer Setup

## Framework

This project uses Vue 3 + Vite and frequently implements Vue 3's `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more. For styling, this project uses Tailwind CSS; [docs found here](https://tailwindcss.com/).

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/)
- [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
- [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)

## Getting Setup

- Once pulled, open a terminal in VS Code and run `npm i`.
- You may wish to update out-of-date packages using `npm audit fix`.
- To spin up the project locally use `npm run dev`.
